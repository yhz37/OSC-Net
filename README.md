# OSC-Net: A Multi-Fidelity Machine Learning Model for Organic Solar Cell

Official implementation of **OSC-Net**, a multi-fidelity machine learning framework developed to predict the power conversion efficiency (PCE) of organic solar cells (OSCs) by integrating low-fidelity computational data and high-fidelity experimental measurements.

This repository accompanies the paper:

**Yang, Haizhou; Wold, Adam; Ou, Junlin; Rech, Jeromy J.; You, Wei; Wang, Yi**  
**"OSC-Net: A Multi-fidelity Machine Learning Model for Organic Solar Cell"**  
*Journal of Materials Chemistry A*, 2025  
DOI: 10.1039/D5TA08724D
